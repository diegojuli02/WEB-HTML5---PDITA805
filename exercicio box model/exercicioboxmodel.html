Box Model

Entendendo a estrutura das caixas no CSS

O Box Model é o conceito que define como os elementos HTML são representados como caixas retangulares no CSS. Ele inclui as propriedades: margem (margin), borda (border), preenchimento (padding) e o conteúdo (content).
